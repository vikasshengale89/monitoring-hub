🚀 Just shipped **monitoring-hub** for the Google Antigravity PromptWars hackathon!

Ever been frustrated by CORS blocking your API tests in the browser? I built a solution. 🛠️

**monitoring-hub** is a premium "Safe-Pass" API Proxy and Monitoring Hub built with the cutting-edge **Angular 19** and **Node.js**. It allows developers to bypass browser restrictions while tracking the health and performance of their APIs in real-time.

**Key Features:**
✅ **Secure API Proxy:** Seamlessly test external APIs without CORS headaches.
✅ **Real-time Health Dashboard:** Instant stats on success rates and response times.
✅ **Zoneless Performance:** Leveraging Angular 19's new zoneless change detection and Signals for a lightning-fast UI.
✅ **WOW Factor: Adaptive Theming:** The UI isn't just pretty; it's alive! It dynamically shifts its theme based on your local time—from a warm sunrise glow to a sleek starry night mode. 🌅🌌
